# Egg-The-Programming-Language-UI
This is is a UI version of the
Egg The Programming Language UI

A simple UI-based implementation of the Egg programming language.

Getting Started
1. Open the `index.html` file in a web browser.
2. Enter your Egg code in the text area.
3. Click the "Run" button to execute the code.
4. The output will be displayed below the button.

Usage
- Enter Egg code in the text area, such as:
print("Hello, World!")

- Click the "Run" button to execute the code.
- The output will be displayed below the button.

Features
- Basic arithmetic operations
- Conditional statements (if)
- Loops (while)
- Functions
- Print statements

Special Forms
- `if`: Conditional statement
- `while`: Loop
- `do`: Evaluate multiple expressions
- `define`: Define a variable or function

Operators
- `+`: Addition
- `-`: Subtraction
- `*`: Multiplication
- `/`: Division
- `==`: Equality
- `<`: Less than
- `>`: Greater than

Example Code
- Print "Hello, World!":
print("Hello, World!")

- Add two numbers:
print(+(2, 3))

- Define a variable:
define(x, 5)
print(x)

- Use an if statement:
if(true, print("True"), print("False"))

- Use a while loop:
define(count, 0)
while(<(count, 5), do(print(count), define(count, +(count, 1))))
